# Contributing

感谢参与“键盘星球”。这是一个面向 6-8 岁儿童的离线优先打字练习项目，贡献应优先保持简单、可访问和可在本地独立运行。

## 本地开发

```bash
npm ci
npm run dev
```

提交前运行：

```bash
npm test
npm run build
npm audit --audit-level=high
```

## 贡献约定

- 新行为先补测试，再实现功能。
- 不引入远程字体、图片、音频、统计服务或账号系统，除非先讨论并更新项目边界文档。
- 输入内容和练习进度保持本地处理，不提交真实用户数据、密钥或 `.env` 文件。
- UI 改动需要检查键盘操作、屏幕阅读器名称和 390px 移动端布局。
- Pull Request 请说明改动目的、验证命令和仍存在的限制。

## 提交方式

1. 从 `main` 创建短期功能分支。
2. 保持提交小而聚焦，使用 `feat:`、`fix:`、`test:`、`docs:` 等前缀。
3. 推送分支并创建 Pull Request，等待 CI 通过后再合并。
